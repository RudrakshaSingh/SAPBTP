# Python + RDBMS + SQL 12-Hour Colab Notebook Package

This package contains three standalone Google Colab notebooks.

## Notebook 1
**RDBMS Foundations, Schema, Keys and Basic SQL**  
Approximate duration: 4 hours  
Covers RDBMS concepts, schema design, keys, constraints and CRUD basics.

## Notebook 2
**Deep Dive into SQL Queries, Joins and Advanced SQL**  
Approximate duration: 4 hours  
Covers joins, aggregation, subqueries, CTEs, CASE, window functions, views, indexes and transactions.

## Notebook 3
**Python Database Integration and Final Project**  
Approximate duration: 4 hours  
Covers Python SQLite integration, CRUD functions, reporting, CSV export, transactions and final project.

## How to use
1. Open Google Colab.
2. Upload one `.ipynb` notebook.
3. Run cells from top to bottom.
4. Each notebook is standalone and creates its own SQLite database.
